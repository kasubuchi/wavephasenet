WavePhaseNet

A DFT-Based Method for Constructing Semantic Conceptual Hierarchy Structures (SCHS)

01 // ABSTRACT

論文の概要

本論文は、Transformer / Attentionメカニズムを測度論と周波数解析の観点から再定式化し、 ハルシネーションの構造的必然性を理論的に解明する。 埋め込み空間はσ-加法族上の条件付き期待値として機能しており、 これが意味的真値集合と同型でないことが論理的一貫性の破綻の根本原因であることを示す。

この観察をもとに、DFTを用いて埋め込み空間内に意味的概念階層構造（SCHS）を明示的に構築する 手法 WavePhaseNet を提案する。 低周波成分がグローバルな意味・意図に、高周波成分がローカルな構文・表現に対応することで、 対角化された空間での精密な意味操作が可能となる。

02 // THEORY

ハルシネーションの構造的必然性

自己回帰モデルの測度論的定式化

自己回帰モデルは結合分布を条件付き確率の積として分解する。 過去のトークン列 \(X_{<t}\) が生成するボレル集合はσ-加法族 \(\mathcal{F}_t = \sigma(x_1,\ldots,x_t)\) をなし、 埋め込みベクトルは可測空間上のランダム変数として解釈できる。

条件付き期待値 \[ \mathcal{L}^P(\Omega, \mathcal{F}, P) \ni f, \quad \|f\|_p = \left(\int_\Omega |f(\omega)|^p \, dP(\omega)\right)^{1/p} \]

モデル全体はσ-加法族上の条件付き期待値の逐次近似器であり、 その出力は点ごとの真値ではなく確率分布上の平均的最適性（ルベーグ測度：期待値E）である。

ハルシネーションの定義と構造的不可避性

ミンコフスキー不等式 \(\|f+g\|_p \leq \|f\|_p + \|g\|_p\) により、 異なる仮説 \(f_1, f_2\) を \(f = \alpha f_1 + (1-\alpha)f_2\) と線形結合できる。 しかし不完備性により、存在しない命題が生じうる。

ハルシネーションの定義 \[ \text{真値集合} \ T \subset V^*, \quad \text{モデル最適集合} \neq T \]

真値集合は一般に非凸・非線形であるのに対し、 \(L^P\)空間は線形かつ凸の構造を持つ。 この「非線形な真値集合 vs 線形・凸な表現空間」のミスマッチが、 ハルシネーションの本質的原因である。

03 // METHOD

WavePhaseNet の手法

周波数帯域と意味の対応

DFTをトークン列方向に適用すると、意味情報は周波数帯域に分解される。

〰️
低周波 (Low-freq)
グローバルなトピック・要約・意図。全ポジションに均一に寄与するDC成分（k=0）
〜〜〜
中周波 (Mid-freq)
談話構造・意味的推移。文レベルの多段階構造
∿∿∿∿∿
高周波 (High-freq)
局所的な語順・語用・表現の違い。位相が位置的差異を決定

DFTの数式定義

シーケンス方向DFT \[ \hat{X}_k = \sum_{n=0}^{N-1} x_n \, e^{-2\pi i k n / N}, \quad k = 0, \ldots, N-1 \]
振幅・位相分解 \[ \hat{X}_k = A_k \odot e^{i\Phi_k}, \quad A_k = |\hat{X}_k| \in \mathbb{R}^d, \quad \Phi_k = \angle\hat{X}_k \in \mathbb{R}^d \]

モデルアーキテクチャ

1
EMBEDDING LAYER
トークン列 → 埋め込み行列 \(X \in \mathbb{R}^{N \times d}\)（系列長N × 埋め込み次元d）
2
DFT LAYER (FFT)
シーケンス方向にFFT適用 → \(\hat{X} \in \mathbb{C}^{N \times d}\)（計算量 \(O(N \log N)\)）
3
BAND SEPARATION
低〜高周波への多段階分離。低周波セット \(L = \{k : |k| \leq K_\text{low}\}\)
4
LOW-FREQ MODULE (Intent)
位相Φと振幅Aを入力として意図ベクトル \(S \in \mathbb{R}^m\) を生成。位置的系列情報を保持
5
HIGH-FREQ MODULE (Generation)
条件付き生成器 \(G\) が \(\hat{X}'_H \sim p(\hat{X}_H | S)\) を出力。normalizing flowやVAEが選択肢
6
INVERSE DFT
\(\hat{X}_L + \hat{X}'_H\) を逆変換し時系列埋め込み行列を復元。PyTorchで軽量実装可能

周波数マスクとしての意味演算子

DFTの本質的価値は可逆変換そのものでなく、周波数依存の意味演算子の挿入にある：

X（入力）
→

→

→
iDFT
→
X̃（変換後）
意味演算子 \[ \tilde{X} = F^{-1} M(\omega) F X \] \[ M(k) = \begin{cases} 1.0 & k=0 \text{（グローバルトピック：保持）} \\ 0.5 & k=1,3 \text{（表現・トーン：減衰）} \\ 0.0 & k=2 \text{（否定・対比構造：除去）} \end{cases} \]
04 // RESULTS

実験結果・理論的根拠

GPT-4（24,576次元）の1/fスペクトル構造

自然言語はZipf則に従い自己相似構造を持つ。 埋め込み振幅のエネルギースペクトルは以下の1/f型減衰を示す：

\[ E(f) \propto \frac{1}{f^\alpha}, \quad \alpha \approx 1.1 \sim 1.4 \]

\(\alpha > 1\) となる理由は、構文制約の存在（文法はセマンティクスより局所的）と TransformerによるLayerNorm/Attentionが高周波を正則化するためである。

コホモロジー正則化と統一最適化目的関数

05 // RELATED WORK

関連研究との比較

類似点
FNet（Google Research）
AttentionをFourier変換で置換することで高速化と意味解析を実現。トークン列に対するDFT適用による意味階層化の有効性を実証済み。本提案のSCHS構築の動機の一つ。
差別化点
WavePhaseNet vs FNet
FNetは複素表現の虚部を破棄するため、共参照関係や文脈依存性の処理が弱い。本提案は位相情報を保持し、意味階層への操作を明示的に導入する点で本質的に異なる。
超越点
RAGベースアプローチとの比較
RAGは強い依存性と拡張性の低さを持つ。WavePhaseNetは埋め込み空間内にSCHSを直接構築することで、ユーザが高度なプロンプト表現スキルを持つ必要性を大幅に低減する。
06 // CONCLUSION

結論

本論文はLLMのハルシネーションを数学的構造から必然的に生じる現象として定式化し、 DFTベースのSCHSを埋め込み空間に導入することでその抑制の理論的枠組みを提示した。 Zipf則と1/fスペクトルに基づく累積エネルギー解析により、 24,576次元から3,000次元への削減が正当化されることが本提案の核心的貢献である。

WavePhaseNetは意味的流れを線形代数で厳密に記述することを可能とし、 次世代の推論指向型言語モデルへの道を開く。
